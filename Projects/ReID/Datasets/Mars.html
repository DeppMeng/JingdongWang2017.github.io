Mars

A large scale video dataset for person re-identification

Mars Dataset

MARS (Motion Analysis and Re-identification Set) is an extension of the Market-1501 dataset [43]. During collection, we placed six near-synchronized cameras in the campus of Tsinghua university. There were Five 1,080*1920 HD cameras and one 640*480 SD camera. MARS consists of 1,261 different pedestrians whom are captured by at least 2 cameras.

Given a query tracklet, MARS aims to retrieve tracklets that contain the same ID.

Download

The dataset package is provided on GoogleDrive, or Baidu Disk . The package contains two folders:

  • bbox_train. There are 509,914 bboxes in this folder, belonging to 625 IDs and 8,298 tracklets.
  • bbox_test. There are 681.089 bboxes in this folder (gallery+query), belonging to 636 IDs and 12,180 tracklets.

Naming Rules of the bboxes

  • In bbox "0065C1T0002F0016.jpg", "0065" is the ID of the pedestrian. "C1" denotes the first camera (there are totally 6 cameras). "T0002" means the 2th tracklet. "F016" is the 16th frame within this tracklet. For the tracklets, their names are accumulated for each ID; but for frames, they start from "F001" in each tracklet.
  • Using the same convention as Market-1501, ID = "00-1" means junk images which do not affect retrieval accuracy; ID = "0000" means distractors, which negatively affect retrieval accuracy.
